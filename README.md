## Configuring
---
Main Config File: `.env`

Copy `.env.example` as `.env` and edit its values to configure the client

### Configuration Options
| Option        | Type   | Default   | Required | Description                                                                                                                                                                                                                                                                                                      |
|---------------|--------|-----------|----------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| WS_PASSWORD   | string | null      | Yes      | The password to connect to the OBS WebSocket.  You can copy this password to your clipboard by navigating to **Tools** -> **WebSocket Server Settings** -> **Show Connect Info**  *If this is your first time using the WebSocket, you may need to enable it and generate a password in the same settings menu.* |
| WS_HOSTNAME   | string | 127.0.0.1 | Yes      | The host to connect to the OBS WebSocket.  Defaults to your local machine.  **Only change this if you have a reason to**                                                                                                                                                                                          |
| WS_PORT       | number | 4455      | Yes      | The port to connect to the OBS WebSocket.  Defaults to `4455`  **Only change this if you also change the WebSocket port within OBS**                                                                                                                                                                             |
| HTTP_PORT     | number | 8181      | Yes      | The port on which the client can be accessed.  Defaults to `8181`.  **Only change this if the default port is currently being used**                                                                                                                                                                              |
| TOWERS_WS_URI | string | null      | No       | This option allows you to connect to a custom WebSocket server to enable extra actions **with your own code**.  Current Extras: `/timer`                                                                                                                                                                         |
